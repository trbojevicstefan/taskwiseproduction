# **App Name**: TaskWiseAI

## Core Features:

- Secure Authentication: User authentication via Email/OAuth using Firebase Auth for secure access.
- Chat Interface: Real-time chat interface for users to input tasks and receive AI-generated task suggestions.
- AI Task Extraction: AI-powered task extraction from chat messages, utilizing a LLM tool to parse user prompts and suggest tasks.
- Task Board: Kanban-style task board to visualize and manage tasks, categorized by status.
- Task Scheduling: Task scheduling and recurrence settings with a clear UI for setting deadlines and repetition.
- Reporting Dashboard: Reporting dashboard to track task completion rates, average latency, and other performance metrics.
- n8n Integration: Integration with n8n for task automation, using webhooks to trigger workflows and update task statuses.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB) for a calm, organized feel.
- Background color: Light gray (#F0F8FF) for a clean, uncluttered look.
- Accent color: Soft orange (#FFB347) for highlighting important actions and elements.
- Body and headline font: 'Inter' sans-serif for a modern, neutral, readable interface.
- Use clean, minimalist icons to represent task categories and actions.
- Implement a clear, intuitive layout with a focus on ease of navigation and task management.
- Incorporate subtle animations to enhance user experience and provide visual feedback.
