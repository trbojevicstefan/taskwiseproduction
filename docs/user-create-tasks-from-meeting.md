# Create Tasks From A Meeting

Use Home or Review Tasks to create a task list without visiting Settings first.

## Paste Notes Or A Transcript

1. Open Home or Review Tasks.
2. Choose Paste notes.
3. Paste meeting transcript, rough notes, or action items.
4. Choose Process notes.
5. Wait for Taskwise to create the meeting and suggested task list.
6. Review opens automatically when processing completes.
7. Approve selected tasks when they are ready for the board.

## Try Sample Data

Use Try sample when you want to test the workflow without connecting an external account.

Sample meetings are intended for evaluation. Delete the sample meeting when you are done testing.

## Connect A Meeting Source

Use Connect Fathom from Home, Review Tasks, or Settings -> Integrations. After connection, sync meetings, review suggested tasks from the Review Tasks queue, then approve the tasks that should appear on the board.
