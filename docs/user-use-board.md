# Use The Board

The Board shows approved workspace tasks by status.

## Open The Board

Use the Board item in the sidebar. If the workspace is still loading, the Board link stays disabled until a valid workspace ID is available.

## Work With Tasks

On the board you can:

- Drag tasks between stages.
- Edit title, description, owner, priority, due date, and status.
- Filter by owner, status, priority, due date, or search text.
- Create new tasks manually.
- Use Task Sweep to find stale or vague tasks that need cleanup.

Meeting tasks appear here after they are approved from Review Tasks or moved to a board from task details.

## Board Recovery

If you open a board URL before workspace context is ready, Taskwise shows a workspace recovery screen instead of routing to `/workspaces/unknown/board`.
