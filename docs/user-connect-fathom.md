# Connect Fathom

Fathom is the primary meeting-source integration for syncing recordings, transcripts, and meeting metadata into Taskwise.

## Connect

1. Open Settings -> Integrations.
2. Find Fathom.
3. Choose Connect.
4. Complete the Fathom OAuth flow.
5. Return to Settings and confirm the Fathom card shows Connected.

## Manage Connections

Admins can open Fathom details to inspect connection status, labels, managed webhooks, and connection-specific actions.

If a connection needs attention, use Fix connection or reconnect from the integration card.

## Sync Meetings

After Fathom is connected, use Home or the meetings list to sync meetings by range. New meetings appear in Review Tasks.
